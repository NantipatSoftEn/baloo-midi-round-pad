# Baloo Midi Round Pad
Web based midi controller with a circular shape.


![screenshot](/images/screenshot.png?raw=true)
# Development Stacks
- React
- Web MIDI API
- Stylus
- Css modules

# Features
- Full mode scale selection.
- Change octave automatically.
- Root changing.

# Developing
```
  npm run dev
```

# Building
```
  npm run build
```
